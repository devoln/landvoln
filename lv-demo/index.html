Minimal, readable HTML for static landings — no build step.

What is LandVoln?

LandVoln is an experimental micro-framework for building static landing pages using custom HTML tags (a tiny DSL) and lightweight content.

Why it exists

You want a landing page that is easy to read and edit in raw HTML. You don’t want to ship a SPA for a simple marketing page. You want reusable blocks without setting up a build pipeline. You need just a bit of interactivity (like a carousel) with minimal JS.

Key features

Custom HTML DSL
Tags like <lv-section> and <lv-carousel> keep markup declarative.
Minimal JS
Most blocks work without JS. Only interactive pieces enhance with tiny scripts.
Themes via CSS vars
Swap a single theme file to change the look.
Static deploy
Copy files to any hosting or publish with GitHub Pages.
Progressive enhancement
Carousel gracefully degrades without JS.
Fork-friendly
Edit plain HTML + CSS. No tooling required.

Getting started

1. Fork the repository

Keep `framework/` and create your own landing folder next to `lv-demo/`.

2. Copy the demo HTML

Replace text blocks and update socials/CTA links.

3. Pick a theme

Use `theme1` or `theme2`, or create your own theme file with CSS variables.
